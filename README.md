## Categories

| Category | Description |
|----------|-------------|
| **Web** | Web application vulnerabilities — SQL injection, XSS, SSRF, IDOR, authentication bypasses, deserialization. |
| **Crypto** | Cryptographic weaknesses — weak ciphers, padding oracles, hash collisions, improper entropy. |
| **AI/LLM** | Attacks against large language models and AI systems — prompt injection, jailbreaking, training data extraction, model inversion, indirect injection. |
| **Pwn** | Binary exploitation — buffer overflows, format string attacks, ROP chains, heap exploitation. |
| **Forensics** | Digital forensics — memory dumps, packet captures, steganography, log analysis, file carving. |
| **Reverse Engineering** | Static and dynamic analysis of binaries — disassembly, decompilation, anti-debug bypassing, firmware analysis. |
